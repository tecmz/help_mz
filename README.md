# 魔众帮助中心系统

魔众帮助中心系统基于PHP开发，支持支持快捷导航和常见问题，帮助您搭建一套全面的帮助中心系统。

## 运行环境


```
操作系统
Linux/Unix 或&nbsp;&nbsp;Windows
软件环境
Laravel 5.1的运行环境
Apache/Nginx , PHP 5.5.9+ / PHP 7.0 , MySQL 5.0+
```
